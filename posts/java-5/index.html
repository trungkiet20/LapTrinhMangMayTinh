Java: Làm việc với file và I/O

October 17, 2025

Trong lập trình, thao tác đọc và ghi dữ liệu từ file là nhu cầu rất phổ biến. Java cung cấp nhiều API mạnh mẽ cho việc xử lý file, đặc biệt là gói java.io và java.nio.file. Bài viết này sẽ giới thiệu các ví dụ cụ thể về đọc/ghi file, lưu ý về encoding, và cách quản lý tài nguyên hiệu quả với try-with-resources.

1. Đọc file bằng BufferedReader

BufferedReader giúp đọc file văn bản theo dòng, hiệu quả hơn FileReader nhờ có bộ nhớ đệm.


import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class ReadFileExample {
    public static void main(String[] args) {
        String filePath = "example.txt";

        try (BufferedReader reader = new BufferedReader(new FileReader(filePath))) {
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println(line);
            }
        } catch (IOException e) {
            System.out.println("Đã xảy ra lỗi khi đọc file: " + e.getMessage());
        }
    }
}

Lưu ý: khối try-with-resources tự động đóng BufferedReader sau khi hoàn tất, tránh rò rỉ tài nguyên.

2. Ghi file bằng BufferedWriter

Tương tự, ta có thể ghi dữ liệu vào file bằng BufferedWriter hoặc Files.write().


import java.io.BufferedWriter;
import java.io.FileWriter;
import java.io.IOException;

public class WriteFileExample {
    public static void main(String[] args) {
        String filePath = "output.txt";

        try (BufferedWriter writer = new BufferedWriter(new FileWriter(filePath))) {
            writer.write("Xin chào thế giới!");
            writer.newLine();
            writer.write("Ghi file bằng Java thật đơn giản.");
        } catch (IOException e) {
            System.out.println("Lỗi khi ghi file: " + e.getMessage());
        }
    }
}

File output.txt sẽ được tạo (hoặc ghi đè) và chứa nội dung được viết trong chương trình.

3. Đọc và ghi với java.nio.file.Files

Từ Java 7 trở đi, ta có thể dùng API trong java.nio.file để thao tác nhanh hơn và dễ đọc hơn:


import java.nio.file.*;
import java.io.IOException;
import java.util.List;

public class NIOExample {
    public static void main(String[] args) throws IOException {
        Path path = Paths.get("data.txt");

        // Ghi file
        Files.write(path, List.of("Dòng 1", "Dòng 2", "Dòng 3"));

        // Đọc file
        List<String> lines = Files.readAllLines(path);
        for (String line : lines) {
            System.out.println(line);
        }
    }
}

Ưu điểm của NIO là code ngắn gọn, hỗ trợ Unicode tốt và xử lý nhanh hơn I/O truyền thống.

4. Lưu ý về encoding

Khi làm việc với file có tiếng Việt hoặc ký tự đặc biệt, nên chỉ định encoding rõ ràng (ví dụ UTF-8):


import java.nio.charset.StandardCharsets;
import java.nio.file.*;
import java.util.List;

public class EncodingExample {
    public static void main(String[] args) throws Exception {
        Path path = Paths.get("utf8.txt");
        Files.write(path, List.of("Xin chào!"), StandardCharsets.UTF_8);

        List<String> lines = Files.readAllLines(path, StandardCharsets.UTF_8);
        System.out.println(lines.get(0));
    }
}

Điều này giúp tránh lỗi hiển thị chữ “rác” khi đọc ghi văn bản có dấu.

5. Kết luận

Java cung cấp nhiều cách linh hoạt để thao tác với file: từ I/O truyền thống đến NIO hiện đại. Kết hợp với try-with-resources giúp mã nguồn gọn gàng và an toàn hơn. Hiểu rõ encoding cũng giúp xử lý dữ liệu văn bản chính xác, đặc biệt khi làm việc với tiếng Việt.
