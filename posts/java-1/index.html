Java: Kiến thức cơ bản và môi trường phát triển

October 17, 2025

Java là một trong những ngôn ngữ lập trình phổ biến và mạnh mẽ nhất thế giới. Được phát triển bởi Sun Microsystems (nay thuộc Oracle), Java nổi tiếng với khẩu hiệu “Write once, run anywhere” — viết một lần, chạy ở mọi nơi.

1. Cài đặt môi trường phát triển Java

Trước khi viết chương trình, bạn cần cài đặt JDK (Java Development Kit). JDK bao gồm:

  • JRE (Java Runtime Environment): môi trường để chạy chương trình Java.
  • javac: trình biên dịch file .java thành bytecode .class.
  • Công cụ CLI: như java, jar, javadoc.

Tải JDK tại: Adoptium hoặc Oracle.

java -version
javac -version

Nếu các lệnh hiển thị phiên bản, bạn đã cài thành công.

2. Sử dụng IDE để lập trình Java

Bạn có thể viết Java bằng trình soạn thảo, nhưng dùng IDE giúp dễ dàng hơn. Một số IDE phổ biến:

  • IntelliJ IDEA: mạnh, hỗ trợ nhiều framework.
  • Eclipse: miễn phí, quen thuộc với sinh viên.
  • VS Code: nhẹ, tiện lợi cho dự án nhỏ.

3. Cấu trúc chương trình Java cơ bản

Chương trình Java thường chứa một lớp và phương thức main():

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Xin chào, thế giới Java!");
    }
}

Để chạy:

javac HelloWorld.java
java HelloWorld

Kết quả:

Xin chào, thế giới Java!

4. Quy trình hoạt động của Java

  1. Biên dịch: file .java được biên dịch thành .class (bytecode).
  2. Chạy: JVM đọc bytecode và thực thi trên mọi hệ điều hành.

5. Một số lỗi cơ bản

  • Cannot find symbol: sai tên biến hoặc phương thức.
  • Class not found: chưa biên dịch hoặc sai đường dẫn.
  • Missing return statement: quên trả giá trị trong phương thức có kiểu dữ liệu.

6. Tổng kết

Bạn đã học cách:

  • Cài đặt và kiểm tra JDK.
  • Viết chương trình đầu tiên trong Java.
  • Hiểu quy trình biên dịch và thực thi trong JVM.

Trong bài tiếp theo, chúng ta sẽ tìm hiểu Lập trình hướng đối tượng (OOP) – lớp, đối tượng và thuộc tính trong Java.
