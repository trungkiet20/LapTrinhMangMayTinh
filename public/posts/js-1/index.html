JavaScript: Tổng quan và cách thêm script vào trang web

October 17, 2025

JavaScript là ngôn ngữ lập trình giúp trang web trở nên sống động — cho phép bạn xử lý sự kiện, thay đổi nội dung HTML, tạo hiệu ứng động, và tương tác với người dùng.

1️⃣ JavaScript dùng để làm gì?

  • Thay đổi nội dung HTML và CSS theo thời gian thực.
  • Xử lý sự kiện người dùng (click, gõ phím, di chuột, ...).
  • Tạo các ứng dụng web động như form kiểm tra, slider, game, v.v.
  • Tương tác với server qua API (AJAX / Fetch API).

2️⃣ Cách thêm JavaScript vào trang HTML

Cách 1: Viết trực tiếp trong thẻ <script>

<!DOCTYPE html>
<html>
  <head>
    <title>Ví dụ JavaScript</title>
  </head>
  <body>
    <h1>Xin chào JavaScript!</h1>

    <script>
      alert("Chào mừng bạn đến với trang web của mình!");
    </script>
  </body>
</html>

👉 Khi trình duyệt tải đến thẻ <script>, mã JavaScript bên trong sẽ được thực thi ngay.

Cách 2: Tách file JavaScript riêng

<!-- index.html -->
<script src="main.js"></script>

// main.js
console.log("Trang đã được tải!");

✅ Ưu điểm: Dễ quản lý, tái sử dụng và bảo trì mã.

3️⃣ Thao tác DOM cơ bản

DOM (Document Object Model) cho phép JavaScript truy cập và thay đổi nội dung HTML.

<h2 id="title">Xin chào</h2>
<button id="btn">Đổi nội dung</button>

<script>
  const btn = document.getElementById("btn");
  const title = document.getElementById("title");

  btn.addEventListener("click", () => {
    title.textContent = "Nội dung đã được thay đổi!";
  });
</script>

🧠 Ở ví dụ này, ta:

  • Lấy phần tử bằng getElementById()
  • Thêm lắng nghe sự kiện bằng addEventListener()
  • Thay đổi nội dung bằng thuộc tính textContent

4️⃣ Một số cách khác để truy xuất phần tử

  • document.querySelector() – Chọn phần tử đầu tiên khớp với CSS selector.
  • document.querySelectorAll() – Lấy tất cả phần tử khớp selector.
  • document.getElementsByClassName() – Lấy theo class name.
const btn = document.querySelector("#btn");
const items = document.querySelectorAll(".item");

5️⃣ Gợi ý: Chèn script cuối body

Để tối ưu tốc độ tải trang, nên đặt thẻ <script> trước thẻ </body>, giúp HTML hiển thị trước khi JS được tải.
