JavaScript: Xử lý sự kiện và tạo tương tác đơn giản

October 17, 2025

Trong lập trình web, sự kiện (event) là yếu tố cốt lõi để trang web trở nên “sống động”. Chúng cho phép người dùng tương tác với trang — nhấn nút, nhập dữ liệu, di chuột, hoặc gửi form. Bài viết này hướng dẫn bạn tạo một form đơn giản, bắt sự kiện, kiểm tra dữ liệu và hiển thị kết quả ngay trên trang HTML tĩnh.

1. Giới thiệu về sự kiện trong JavaScript

Mỗi hành động của người dùng trên trang web (click, nhập chữ, di chuột, v.v.) đều kích hoạt một sự kiện. JavaScript cho phép bạn “lắng nghe” (listen) và “xử lý” (handle) các sự kiện đó bằng cách gắn event listener vào phần tử HTML.

2. Ví dụ: Form tương tác với người dùng

Giả sử bạn muốn tạo một form nhập tên, sau khi người dùng nhấn nút “Gửi”, trang sẽ hiển thị lời chào.


<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <title>Xử lý sự kiện cơ bản</title>
</head>
<body>
  <h2>Nhập tên của bạn</h2>
  <form id="nameForm">
    <input type="text" id="nameInput" placeholder="Nhập tên...">
    <button type="submit">Gửi</button>
  </form>
  <p id="message"></p>

  <script>
    const form = document.getElementById('nameForm');
    const input = document.getElementById('nameInput');
    const message = document.getElementById('message');

    form.addEventListener('submit', function(event) {
      event.preventDefault(); // Ngăn form reload trang
      const name = input.value.trim();

      if (name === '') {
        message.innerText = 'Vui lòng nhập tên trước khi gửi!';
        message.style.color = 'red';
      } else {
        message.innerText = `Xin chào, ${name}! Rất vui được gặp bạn.`;
        message.style.color = 'green';
      }
    });
  </script>
</body>
</html>

3. Giải thích

  • addEventListener('submit', ...) giúp gắn sự kiện vào form khi người dùng nhấn nút gửi.
  • event.preventDefault() ngăn hành vi mặc định (reload trang khi submit).
  • innerText dùng để thay đổi nội dung hiển thị trong phần tử HTML.
  • Ta có thể chỉnh màu chữ, CSS trực tiếp qua element.style.

4. Mở rộng: Xử lý nhiều loại sự kiện khác

Bạn có thể lắng nghe các sự kiện khác như:


button.addEventListener('click', function() { ... });
input.addEventListener('input', function() { ... });
window.addEventListener('load', function() { ... });

JavaScript có thể bắt hàng trăm loại sự kiện khác nhau — từ bàn phím, chuột, cảm ứng đến bàn phím ảo hoặc camera.

5. Kết luận

Việc hiểu và sử dụng event giúp bạn tạo ra trải nghiệm tương tác mượt mà hơn cho người dùng. Kết hợp với thao tác DOM, bạn có thể làm form động, popup, hoặc hiệu ứng tùy chỉnh mà không cần framework phức tạp.
